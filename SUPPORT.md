# Support

## How to Get Help

**Have a question about using this skill?**
Open a [GitHub Discussion](https://github.com/henriquesantanati/linkedin-profile-score/discussions) (once enabled) or start a conversation in the Issues tab with the label `question`.

**Found a bug or incorrect scoring?**
Open a [bug report](https://github.com/henriquesantanati/linkedin-profile-score/issues/new?template=bug_report.md).

**Want to suggest an improvement?**
Open a [feature request](https://github.com/henriquesantanati/linkedin-profile-score/issues/new?template=feature_request.md).

## What This Project Does NOT Cover

- Writing your LinkedIn content for you (this is an audit tool)
- Job application tracking or strategy
- LinkedIn API integration or automation
- Advice specific to a particular industry or role (the scoring is general-purpose)

## Response Times

This is a solo-maintained open source project. Expect responses within 7 days for issues and PRs. If something is urgent, mention it in the issue title.
